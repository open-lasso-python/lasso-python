DimredRun
---------

This class can be used in order to comfortably represent multiple D3plot samples in 3D.
It offers functions to:

* subsample
* reduce order
* cluster
* output 3D plot as webpage

For ease of use, a console commandline tool is provided with the :doc:`run`
